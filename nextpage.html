BASIC TERMINOLOGIES

  • WWW
    • IT STANDS FOR "WORLD WIDE WEB". IT IS AN INTERCONNECTED SYSTEM OF PUBLIC WEBPAGES ACCESSIBLE THROUGH THE INTERNET.

  • URL
    • IT STANDS FOR "UNIFORM RESOURCE LOCATOR". IT IS A UNIQUE IDENTIFIER USED TO LOCATE A RESOURCE ON THE INTERNET.IT IS ALSO REFFERED AS "WEB ADDRESS".

  • SERVER
    • A SERVER IS A COMPUTER PROGRAM OR DEVICE THAT PROVIDES A SERVICE TO ANOTHER COMPUTER PROGRAM AND ITS USER,ALSO KNOWN AS THE CLIENT.

  • WEB SERVER
    • WEB SERVER IS A COMPUTER THAT RUNS WEBSITES.IT IS A TYPE OF SOFTWARE THAT SEARCHES,FINDSAND PROVIDES DOCUMENTS TO THE BROWSERS,AS REQUESTED BY THEM.

  • WEB BROWSER
    • A SOFTWARE APPLICATION USED TO ACCESS INFORMATION ON THE WORLD WIDE WEB.WHEN A USER REQUESTS A WEB PAGE FROM A PARTICULAR WEBSITE,THE WEB BROWSER RETRIVES THE NECESSARY CONTENT FROM A WEB SERVER AND THEN DISPLAYS THE PAGE ON THE USER'S DEVICE.

  • WEB PAGE
    • IT IS A SINGLE HYPETEXT DOCUMENT AVAILABLE ON WWW.EACH WEBPAGE IS LINKED WITH A UNIQUE URL,TWOPAGES CANNOT HAVE THE SAME URL.

  • WEBSITE
    • IT IS A COLLECTION OF WEB PAGES AND RELATED CONTENT THAT IS IDENTIFIED BY A COMMON DOMAIN NAME AND PUBLISHED ON ATKLEAST ONE WEB SERVER.

  • STATIC WEBSITE
    • A STATIC WEBSITE IS ONE WITH STABLE CONTENT,WHERE EVERY USER SEES THE EXACT SAME THING ON EACH INDIVIDUAL PAGE.

  • DYNAMIC WEBSITE
    • DYNAMIC WEBSITEIS ONE THAT CAN DISPLAY DIFFERENT CONTENT AND PROVIDE USER INTERACTION.IT ALLOWING ITS CONTENT TO CHANGE WITH THE USER.
