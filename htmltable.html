HTML TABLE TAG

HTML table tag is used to display data in tabular form (row * column). There can be many columns in a row.We can create a table to display data in tabular form, using < table> element, with the help of < tr> , < td>, and < th> elements. Each table, table row is defined by < tr> tag, table header is defined by < th>, and table data is defined by < td> tags.

"< table> It defines a table."
"< tr> It defines a row in a table."
"< th> It defines a header cell in a table."
"< td> It defines a cell in a table."
"< caption> It defines the table caption."
"< tbody> It is used to group the body content in a table."
"< thead> It is used to group the header content in a table."
"< tfooter> It is used to group the footer content in a table."

EXAMPLE:

"< table border="1"> < tr>"
"< th>First_Name< /th>"
"< th>Last_Name< /th>"
"< th>Marks< /th>"
"< /tr>"
"< tr>"
"< td>Sonu< /td>"
"< td>Jai< /td>"
"< td>60< /td>"
"< /tr>"
"< tr>"
"< td>Ram < /td>"
"< td>kumar< /td>"
"< td>80< /td>"
"< /tr>"
"< tr>"
"< td>Swati< /td>"
"< td>Sharma< /td>"
"< td>82< /td>"
"< /tr>"
"< /table>"

"OUTPUT:"

First_Name Last_Name Marks
Komal jain 60
Samar thakur 75
Kanika Verma 83
