LOOPS IN PL/SQL

The PL/SQL loops are used to repeat the execution of one or more statements for specified number of times. These are also known as iterative control statements.

Syntax OF EXIT LOOP :

LOOP
Sequence of statements;
EXIT;
END LOOP;

  • TYPES OFLOOPS IN PLSQL
    1.EXIT LOOP. 1.WHILE LOOP. 1.FOR LOOP. 1.CURSOR FOR LOOP.

Example :

DECLARE
i number := 1;
BEGIN
loop
exit when i>10;
dbms_output.put_line?(i);
i:=i+1;
end loop;
END;
