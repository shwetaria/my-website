  • WHAT IS DATA?
    • It is a collection of facts and figures that can be processed to produce information.

  • WHAT IS DATABASE?
    • Database is a collection of related data.That related data is organised so that it can be easilly accessed,manages and updated.

  • WHAT IS DATABASE MANAGEMENT SYSTEM?
    • Database management system is the software that allows a computer to perform database functions of storing,adding,deleting and modifying data.

  • OBJECTIVES OF DATABASE :
    • 1.Provide for mass storage of relevant data.
    2.Making easy access to data for authorized user.
    3.Providing prompt response to users requests for data.
    4.Eliminate duplicate data.
    5.Serving different types of users.


  • ADVANTAGES OF DBMS :
    • 1.Minimized Redundancy.
    2.Data Security.
    3.Data Integrity.
    4.ELIMINATION OF INCONSISTENCY.
    5.Sharing Of Data.
    6.Provides Backup And Recovery.


  • DISADVANTAGES OF DBMS :
    • 1.High Cost.
    2.Security Threats.
    3.Complexity of Backup And Recovery.
    4.Currency Maintenance.
    5.Frequency Upgrade/Replacement cycles.


  • DATA LANGUAGES
    • It refers to provide a number of operations on the database such as insert,delete,update and retrievesdata without knowing about the physical representation of data to different types of users. It is a language in which the users request for information from the database.It is asimple and powerful language.

    1. DDL -"DATA DEFINITION LANGUAGE"

    # It is used to create , remove and modify the structure of database objects like-tables,views,indexes in database.
    # DDL describes the organization of data that exists or will be exists in a database.


    2. DML- "DATA MANIPULATION LANGUAGE"

    # It enables to manipulate data in database.
    # It is used to retrive, insert , delete , modify database information.


    3. DCL- "DATA CONTROL LANGUAGE"

    # It is used toaccess the data stored in database.
    # We can perform any operations in database such as creating table , sequence andview.
    # It defines 2 commands :-
    * GRANT : It gives user access privilage to database.
    * REVOKE : It takes back permissions from users.
